# Elena Noir Photography Portfolio

A dark-themed portfolio website for photographers featuring an elegant masonry gallery with lightbox functionality.

## Features

- **Dark Elegant Theme** - Sophisticated dark design with gold accents
- **Masonry Gallery** - Responsive grid layout for photo showcase
- **Lightbox Modal** - Full-size image viewing with navigation
- **Keyboard Support** - Arrow keys and Escape for lightbox control
- **Smooth Scrolling** - Seamless navigation between sections
- **Fully Responsive** - Adapts beautifully to all screen sizes

## Setup

1. Open `index.html` in any modern browser
2. Replace placeholder images with your own photography
3. Update text content as needed

## Customization

Edit the `photos` array in the script section to add your own images:

```javascript
const photos = [
    { src: 'your-image-url.jpg', title: 'Image Title' },
    // Add more photos...
];
```
